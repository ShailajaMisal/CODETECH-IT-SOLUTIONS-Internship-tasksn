Projects

Project One

The "Fruit Selling" website is a project developed as part of my college coursework, showcasing and selling fresh fruits. Using HTML and CSS, the site features a clean, responsive design with intuitive navigation and product listings. Key features include a searchable fruit catalog, detailed product pages, and a shopping cart. Achievements include successfully implementing responsive design and creating a user-friendly interface. Challenges faced included optimizing images for faster load times and ensuring cross-device compatibility.

Project Two
